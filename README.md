# FRIDAY

> Personal AI Assistant inspired by Tony Stark's FRIDAY

## Description

FRIDAY is a personal AI assistant designed to help manage finances and day-to-day planning, drawing inspiration from Tony Stark's iconic AI companion.

## Features Planned

- **Expense Tracking** — Monitor and categorize daily expenses
- **EMI Reminders** — Never miss a loan or EMI payment
- **Salary Planning** — Plan and allocate salary across goals and expenses
- **Remittance Recommendations to India** — Smart suggestions for sending money to India at optimal rates and timings
